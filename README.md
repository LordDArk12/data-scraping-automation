# Automated Data Extraction & Structuring Tool

Скрипты для автоматического сбора (веб-скрейпинга), очистки и структурирования данных из открытых источников.

## 🚀 Функционал
* **Сбор данных:** Автоматический парсинг целевых веб-страниц и извлечение контента по заданным селекторам.
* **Очистка данных (ETL):** Обработка сырых данных, удаление дубликатов, заполнение пропусков и форматирование с помощью Pandas.
* **Автоматизация запуска:** Bash-скрипты для автоматизации регулярного запуска парсера по расписанию (Cron/Task Scheduler).
* **Экспорт:** Автоматическая выгрузка структурированной информации в форматы Excel (XLSX) и CSV по готовым шаблонам.

## 🛠 Стек технологий
* **Python:** Requests, BeautifulSoup4, Pandas
* **Automation:** Bash scripting, Linux OS
